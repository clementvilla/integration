Refresh

Projet personnel

Refresh est un projet personnel ayant pour but d'expérimenter les animations de chargement sur Adobe XD.

Cette animation pourrait notamment être utilisé dans un réseau social, où sur une application visant un public aimant les animations provoquant une certaine distraction. L'inspiration afin de créer cette animation m'est venue après une vidéo vue sur YouTube.
